1. template literal `${a}${b}`:문자열을 만드는것 2.Number()함수 문자열을 숫자로 변환 3.Math.max()함수 주어진 함수중에 가장큰값 4.+ 연산자는 불리언 값을 숫자로 변환하는 역할을 합니다. true는 1로, false는 0으로 변환됩니다. function solution(str1, str2) { return +str2.includes(str1); } 5.indexOf(): indexOf() 메서드는 대상 문자열에서 지정된 문자열이 발견된 위치(인덱스)를 반환합니다. 문자열이 발견되지 않으면 -1을 반환합니다. 대소문자를 구분합니다.
