true는 1로, false는 0으로 변환됩니다.
function solution(str1, str2) {

}

5.indexOf():
indexOf() 메서드는 대상 문자열에서 지정된 문자열이 발견된 위치(인덱스)를 반환합니다.
문자열이 발견되지 않으면 -1을 반환합니다.
대소문자를 구분합니다.
ES5에서부터 사용 가능한 메서드입니다.

9.반복 가능한(iterable) 객체는 여러 개의 원소를 포함하고 있으며,
이러한 원소들에 대해 반복적으로 작업할 수 있는 객체입니다.
Python에서 반복 가능한 객체는 다음과 같은 특징을 가지고 있습니다
1.list:순서가 있는 데이터 집합으로 각요소에는 인덱스로 지정되어있습니다
2.tuple: 튜플은 리스트와 유사하지만 불변순서가 있는 데이터 집합
3.string: 문자열은 문자의 시퀀스이면, 각문자에 인덱스가 지정되어있다
4.set: 세트는 고유한 요소들의 집합으로 순서가 없다
5.dictionary:딕셔너리는 키-값 쌍의 집합으로 각키를 사용하여 해당값찾기가능
